# Grok Discord Bot - パフォーマンス最適化

## 応答速度の改善

### 実装済みの最適化
1. **max_tokens削減**: 1024 → 512（応答が短く、速くなります）
2. **temperature調整**: 0.8 → 0.7（より集中した応答）
3. **top_p調整**: 1.0 → 0.9（サンプリング速度向上）
4. **タイピングインジケーター**: 5秒ごとに更新（ユーザーに処理中を通知）

### さらなる高速化（オプション）

#### 1. より小さいモデルを使用
`src/lib/ai.js`の`model`を変更:
```javascript
model: "llama-3.1-8b-instant", // 70Bより高速
```

#### 2. Groq APIキーのレート制限を確認
- 無料プランは制限あり
- 有料プランでより高速に

#### 3. 会話履歴を削減
`src/index.js`で:
```javascript
if (userContext.length > 6) { // 10 → 6に変更
    userContext.splice(0, userContext.length - 6);
}
```

### 応答時間の目安
- **通常チャット**: 2-5秒
- **ファクトチェック**: 3-7秒（プロンプトが長いため）
- **画像分析**: 5-10秒（ビジョンモデル使用）

### トラブルシューティング
- Renderの無料プランは初回起動が遅い（コールドスタート）
- Keep-alive機能で軽減済み
- 頻繁に使用すると速くなります
